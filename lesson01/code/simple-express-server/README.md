npm init -y - инициализирует проект. -y - нужен, чтобы были добавлены настройки по умолчанию.
Команда нужна, чтобы мы могли устанавливать библиотеки. Используется 1 раз в начале

npm install express - установка express

npm i (install) - установка зависимостей на основе package.json

ctrl+c - остановить сервер
